| Boundary | Description |
|----------|-------------|
| OpenAI API → service | LLM returns untrusted JSON that gets parsed and written to the DB |
| service → Debtor/Contact rows | Writes scoped by tenantId; cross-tenant leakage must be prevented |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|

| T-05-02 | Information Disclosure | cross-tenant debtor read/write | mitigate | Every query includes `tenantId` in where-clause; getUnifiedContext/refreshMemory take tenantId as first arg |
| T-05-03 | Denial of Service | OpenAI timeout/rate-limit during refreshMemory | mitigate | LLM call wrapped in try/catch with heuristic fallback; refreshMemory never throws on LLM failure (Landmine 6) |
| T-05-SC | Tampering | npm/pip/cargo installs | accept | No new packages introduced this phase (RESEARCH §Standard Stack: zero new deps) |
</threat_model>
